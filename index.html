Definição de HTML

HTML significa Hypertext Markup Language

É a linguagem padrão para criação de páginas da web. Ele define a estrutura básica de uma página da web e usa uma série de elementos, tags e atributos para formatar o conteúdo.

Estrutura Básica

Em uma estrutura HTML básica poderemos observar as seguintes partes:

  1. HTML: Elemento raiz que engloba todo o conteúdo da página.
  2. HEAD: Contém metadados e informações sobre a página (título).
  3. BODY: Contém o conteúdo principal da página, visível aos usuários.

Listas

HTML suporta dois tipos principais de listas: listas ordenadas e listas não ordenadas.

Listas Ordenadas

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

  • Item A
  • Item B
  • Item C

Títulos

HTML fornece seis níveis de títulos, de h1 a h6, que representam diferentes níveis de importância.
